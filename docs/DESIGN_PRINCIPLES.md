# Design Principles

Debo is designed to feel like a serious productivity tool — not a toy, not a demo, not a template.

## Core Principles

### Minimal

Every element earns its place. Remove anything that does not serve the user. White space is intentional. Complexity is hidden until needed.

### Premium

The app should feel polished and trustworthy. Attention to detail in typography, spacing, borders, and transitions. Users should feel like they are using a professional tool.

### Calm

No flashing animations. No aggressive notifications. No visual noise. Debo is a quiet space for your thoughts. The interface should feel like a clean desk, not a cluttered dashboard.

### Trustworthy

Dark and light themes that feel solid. Subtle borders and rounded corners. Consistent spacing. No cheap effects. The design should communicate: "your data is safe here."

### Keyboard-Friendly

Power users work with keyboards. Every action should be reachable without a mouse. Capture should feel instant — type, Cmd+Enter, done.

### Fast

No loading spinners for local operations. Instant saves. Instant search. The app should never make you wait for something that happens on your own machine.

### Local-First

The design reflects the local-first philosophy. No cloud badges. No sync indicators. No account avatars. Just you and your data.

## Visual Language

### Color

- Neutral backgrounds (dark: `#0f0f0f`, light: `#ffffff`)
- Muted secondary text
- Single accent color used sparingly
- Semantic colors for status only (success, warning, danger)

### Typography

- System font stack (Inter preferred)
- Clear hierarchy: large headings, medium body, small labels
- Generous line height for readability
- Tight letter spacing for headings

### Components

- Rounded cards with subtle borders
- Pill-shaped badges for types
- Ghost buttons for secondary actions
- Solid buttons for primary actions
- Inputs with clear focus states

### Layout

- Fixed sidebar navigation
- Scrollable main content area
- Responsive for smaller windows
- Consistent padding and gaps

## What We Avoid

- Childish gradients or neon colors
- Generic AI robot illustrations or mascots
- Overcomplicated dashboards with charts everywhere
- Skeuomorphism or 3D effects
- Decorative animations that slow the user down
- Dark patterns or engagement tricks
- Cluttered toolbars

## Inspiration

Debo draws inspiration from tools that respect the user:

- Linear — clean, fast, opinionated
- Bear — beautiful note-taking
- Things 3 — calm productivity
- iA Writer — focused writing
- Notion — flexible but organized

The goal is a tool that feels inevitable — like it was always meant to exist in this form.
