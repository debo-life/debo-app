# Changelog

All notable changes to Debo will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## 0.1.0

### Added

- Initial Tauri + React + TypeScript app scaffold
- Memory capture (thoughts, tasks, ideas, links, journal)
- localStorage persistence
- Dashboard with stats and recent memories
- Memory browser with delete support
- Search across title, content, and type
- Task view with completion tracking
- Settings page with data management
- Dark/light theme based on system preference
- Keyboard shortcut: Cmd/Ctrl + Enter to save
- Toast notifications for save feedback
- Responsive sidebar layout
- Community files: LICENSE, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY, GOVERNANCE, ROADMAP
- Documentation: ARCHITECTURE, PROJECT_STRUCTURE, DESIGN_PRINCIPLES, RELEASE_PROCESS, GOOD_FIRST_ISSUES, RFC_TEMPLATE
- GitHub issue templates, PR template, CI workflow
